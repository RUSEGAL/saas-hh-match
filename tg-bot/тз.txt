7. Vacancy Search
7.1 Manual Search Flow
1. User: /vacancies → 🚀 Начать поиск
2. Bot: "Выберите резюме для сравнения"
3. User: Выбирает из списка
4. Bot: "Введите поисковый запрос"
5. User: "Golang developer"
6. Bot: "Дополнительные фильтры?"
   - Тип занятости: Трудоустройство / Самозанятость / ИП
   - Формат: Удалёнка / Офис / Гибрид

9.2 Payment Flow
1. User: /payment → Оплатить
2. Bot: Выбор тарифа:
   - 1 месяц — 299₽
   - 3 месяца — 799₽ (скидка 10%)
   - 12 месяцев — 2499₽ (скидка 30%)
3. User: Выбирает тариф
4. Bot: Генерирует ссылку на оплату (ЮKassa / CloudPayments)
5. User: Оплачивает
6. Webhook от платёжной системы → обновляет статус в DB
7. Bot: Уведомляет пользователя "✅ Подписка активирована!"

14. Database (Bot-specific)
CREATE TABLE bot_users (
    id INTEGER PRIMARY KEY,
    telegram_id INTEGER UNIQUE,
    username TEXT,
    created_at DATETIME,
    is_active BOOLEAN DEFAULT true
);
CREATE TABLE user_schedules (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    cron_expr TEXT,
    resume_id INTEGER,
    query TEXT,
    filters TEXT, -- JSON
    is_active BOOLEAN DEFAULT true,
    last_run DATETIME,
    FOREIGN KEY(user_id) REFERENCES bot_users(telegram_id)
);
