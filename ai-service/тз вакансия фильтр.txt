4. Фильтрация Exclude Words
Для каждой вакансии проверять:
- vacancy.name — название
- vacancy.snippet.preview_text — описание
Если найдено совпадение (case-insensitive) — исключить вакансию.
func containsExcludeWord(text string, excludeWords []string) bool {
    text = strings.ToLower(text)
    for _, word := range excludeWords {
        if strings.Contains(text, strings.ToLower(word)) {
            return true
        }
    }
    return false
}
5. Логика обработки
1. Получить job из NATS
2. Построить URL для hh.ru API с учётом employment_types и work_formats
3. Получить вакансии (limit штук)
4. Для каждой вакансии:
   a. Проверить exclude_words (название + описание)
   b. Если совпадает — пропустить
   c. AI semantic matching с резюме
   d. Если score >= 0.70 — добавить в результат
5. Отправить webhook с результатами
6. Config
HH_API_URL=https://api.hh.ru
HH_API_TIMEOUT=30s
MATCH_SCORE_THRESHOLD=0.70
7. Rate Limiting
- hh.ru: max 2 requests/sec
